description Project Overview

Understanding the inner workings of complex neural networks remains a significant challenge for researchers. The "Black Box" problem often hinders the optimization of model architectures.

I built the Neural Net Visualizer to bridge this gap. This web-based application parses standard model files (ONNX, PyTorch) and generates an interactive 3D graph representation. Users can zoom into specific layers, inspect weights in real-time, and visualize the flow of tensors through the network. The goal was to create a tool that is not only functional but also performant enough to handle models with millions of parameters directly in the browser.

architecture Architecture & System Design

The system follows a client-server architecture designed for heavy computational offloading. The backend handles model parsing and simplification, while the frontend focuses on high-performance WebGL rendering.

psychology_alt Key Challenges & Solutions

speed

Rendering Bottlenecks

Challenge:

Rendering 100k+ nodes for large language models caused the DOM to freeze.

Solution:

Implemented InstancedMesh in Three.js to render identical geometries with a single draw call, maintaining 60 FPS.

dataset

Data Serialization

Challenge:

JSON parsing for 50MB+ model architecture files was too slow on the client.

Solution:

Developed a custom Protocol Buffers schema to stream binary data, reducing payload size by 40% and parse time by 70%.
